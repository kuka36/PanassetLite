# 如何记录银行贷款

PanassetLite 用 **「负债」类资产** 跟踪贷款；余额表示 **剩余本金**（不含未来应付利息）。适用于房贷、消费贷、经营贷等银行贷款。

## 1. 新建负债资产

在 **资产** 页 → **新增资产**：

| 字段 | 说明 |
|------|------|
| 名称 | 如「建设银行房贷」 |
| 类型 | **负债** |
| 币种 | 通常为 CNY |
| 平台 | 放款银行（可选） |

负债资产可用的流水类型：**借入**、**还款**、**估值更新**。

## 2. 核心原则

- **借入**：负债增加，金额为借到的本金。
- **还款**：负债减少，金额只填 **当期偿还的本金**（银行还款明细里会单独列出）。
- **估值更新**：直接设为某日的 **剩余本金**（适合不想逐月拆本金/利息时）。

等额本息、等额本金等还款方式下，**不要把整笔月供记成「还款」**——利息不会减少剩余本金，整笔月供应体现在 **现金类资产的「取出」** 上。

## 3. 两种录入方式

### 方式 A：从当前起算（推荐）

贷款已进行一段时间、只需反映当前状态时：

1. 查银行 App 的 **剩余本金**。
2. 记一笔 **借入**，金额 = 剩余本金，备注可写已还期数、利率等。
3. 之后每月：**还款**（本金部分）+ 在存款账户记 **取出**（整笔月供）。

### 方式 B：补全历史

需要净资产历史曲线更准确时：

1. **借入** 原始贷款金额，时间为 **放款日**。
2. 按银行明细逐月补 **还款**，每条金额为该期 **本金**。
3. 若不想补全部历史，可用 **估值更新** 校正为当前剩余本金，再只维护近期流水。

演示数据（设置 → 加载演示数据）中的房贷示例即采用「借入剩余本金 + 按月还本金」的方式。

## 4. 每月维护示例

假设月供 ¥1,201，其中本金 ¥654、利息 ¥547：

| 资产 | 流水类型 | 金额 | 说明 |
|------|----------|------|------|
| 负债（贷款） | 还款 | ¥654 | 仅本金 |
| 现金存款 | 取出 | ¥1,201 | 整笔月供 |

也可每月查银行 **剩余本金**，用 **估值更新** 直接覆盖，省去拆本金/利息。

## 5. 常见情况

| 情况 | 记法 |
|------|------|
| 提前还款 | **还款**，金额 = 实际减少的本金（非含利息的总还款额） |
| 贷款对应房产 | 另建 **房产实物** 资产记市值；负债单独记，净资产 = 资产 − 负债 |
| 利率/期限变更 | 用 **估值更新** 校正剩余本金，或补一条 **借入**/**还款** 反映本金变动 |

## 6. 计算参考（等额本息）

已知贷款本金 P、年利率 r、期数 n（月），已还 k 期：

- 月供 ≈ `P × [r/12 × (1+r/12)^n] / [(1+r/12)^n − 1]`
- 剩余本金 ≈ `P × [(1+r/12)^n − (1+r/12)^k] / [(1+r/12)^n − 1]`

最终以 **银行账单上的剩余本金与每期本金** 为准；上式仅作录入前估算。
